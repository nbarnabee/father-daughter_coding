Asynchronous Functions

Set reminder

Natively, JS can only "do" one thing at a time. The neat thing about promises and asynchronous programming is that it allows us to overcome that limitation. Other functions (like the one that's causing this text to be displayed) can continue to run while we wait for the timer function to resolve.

This box's appearance was triggered by another asynchronous function. You could also enter data into the forms and submit them while you wait. But without asynchronous programming, that wouldn't be possible.
